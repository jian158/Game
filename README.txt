o摘要
游戏名称：大逃亡
团队名称：用心创造快乐（11组）

    首席设计师：石  楷

游戏概念：这是一款操作简单的类似于雷电系列和雷霆战机的小型2D飞行射击类游戏。这款游戏操作简单，节奏明快，游戏的目的只有一个，便是把出现的敌人完全消灭并击败关卡最后的BOSS就能通关。


o安装说明
本游戏无需安装直接可玩

o操作说明
?当游戏开始后玩家需要通过↑↓←→四个键来移动躲避敌人发射的子弹，并通过**空格键**发射激光炮来攻击敌人，还可以使用**D键**来释放必杀技。
?**Tab键**启动菜单，返回按**enter键**，或提示按键
